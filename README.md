# Odin Recipes

This is my first project from The Odin Project Foundations course.

The website contains a main page with links to three recipes:

* Pizza
* Hamburger
* Ramen

Each recipe page includes an image, description, ingredients and cooking instructions.

## What I Learned

While making this project, I practiced:

* Creating HTML pages
* Using headings, paragraphs and lists
* Adding images and links
* Linking pages using relative paths
* Organizing files into folders
* Using Git and GitHub
